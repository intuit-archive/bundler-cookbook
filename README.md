# bundler cookbook
Configures bundler

# Requirements
* Chef 10.x

Platform:
* CentOS 6.x
* RHEL 6.x

# Usage
Add the bundler cookbook to your role/run_list.

# Attributes

```node["bundler"]["gem_binary"]``` = path to gem binary
```node["bundler"]["version"]``` = version of bundler

# Recipes
## default
Configures bundler
